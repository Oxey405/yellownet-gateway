About YELOWNET

YELLOWNET is a networking protocol made for the Playdate by Oxey405!
It works using WebSerial which allows webpages to communicate with physical devices. YELLOWNET is a standard of communication that
enables the Playdate to communicate with servers around the world by using a Gateway (like this one) to forward packets both ways.

Why not use the Playdate's Wi-Fi chip ?

Unfortunately, the Playdate doesn't just let you have fun with the network as it's considered something sensitive.
Although some games can access online leaderboards and the system has online features, games not published on Catalog cannot access Wi-Fi*. That's why such a complex system is required for YELLOWNET to... well.. work!
* Well you can try and write your own C Networking API implementation for the ESP32 by reverse engineering stuff
BUT note that it's going to be way though and this would probably make Panic mad and we don't want to piss off Panic ♥️
